#To test these test builds out please follow these simple steps:#
#################################################################

1. Click the green button "Clone or Download" in the top right, and then click "Download ZIP"

2. Unzip the downloaded files and throw the entire 'rcm' folder into:
   "Documents\Paradox Interactive\Hearts of Iron IV\mod"
  (if there isn't a "mod" folder just manually create one).

3. Move the file "rcm.mod" or "rcm" (depending on whether the extension is visible) out of this folder into "Documents\Paradox Interactive\Hearts of Iron IV\mod".

4. Select "Eight Years' War of Resistance: Beta Build" in the start up mod menu of HOI4.

5. Enjoy!
